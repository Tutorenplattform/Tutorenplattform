Class: Authentication

tp.auth.service.Authentication

new Authentication($auth, $state, Globals, UserType)

This service is used to interact with the current user session. It is responsible for account login, account logout, permission management and account information retrieval.
Parameters:
Name Type Description
$auth $auth The satellizer service used for all requests concerning JWT token authentication and localStorage token management
$state $state The service used to transition between states
Globals Globals The application's global configuration object
UserType UserType An enumeration containing all types of users
